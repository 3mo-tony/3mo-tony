## 🚀 Featured Work

### 📦 [express-v5-mongo-sanitize](https://github.com/3mo-tony/express-v5-mongo-sanitize)
A published npm package fixing a real breaking change: `express-mongo-sanitize`
stopped working on **Express v5** because `req.query` became read-only. I
rebuilt the sanitization logic to work without mutating `req.query` directly,
keeping NoSQL-injection protection intact for anyone upgrading to Express v5.
→ Built from a real problem, shipped as a public package others can depend on.
